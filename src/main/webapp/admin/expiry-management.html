Setup Instructions
For Automatic Scheduling:
  1. Cron Job (Linux/Mac): Add to crontab: 0 */6 * * * curl -s http://localhost:8080/ShareBite/scheduledTasks.jsp
  2. Windows Task Scheduler: Create task to call: http://localhost:8080/ShareBite/scheduledTasks.jsp
  3. Application Server: Configure scheduler in web.xml or use @Scheduled annotation

Recommended: Run every 6 hours to keep food listings up to date.
